# Maatilayla — Sito Allevamento Barboncini Toy

## Obiettivo
Rebuild del sito WordPress allevamentobarboncinimaatilayla.it in React + Vite,
hostato su Siteground. Stile premium Apple-inspired con glassmorphism.

## Stack
- React 19 + Vite 7 + TypeScript
- Tailwind CSS v4 + @tailwindcss/vite
- React Router v7
- GSAP 3 + Framer Motion + Lenis
- EmailJS (form senza backend)
- Swiper.js + YARL (lightbox)
