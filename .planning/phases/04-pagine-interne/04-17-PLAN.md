**Componente `FaqAccordionSection.tsx`:**

1. **State**: `openId: string | null` ã id della FAQ aperta (null = tutte chiuse)
2. **Import**: faqCategories da faqData.ts, ChevronDown da lucide-react
3. **id="faq"** sulla section (hash scroll dal hero CTA)
4. **Struttura render**:
   - `<section id="faq" className="fa-section">` con sectionRef + IntersectionObserver
   - Per ogni categoria in faqCategories:
     - `<div className="fa-category">`
     - `<h3 className="fa-category-title">{category.title}</h3>`
     - Per ogni item in category.items:
       - `<div className="fa-item" data-open={openId === item.id}>`
       - `<button className="fa-question" onClick={() => toggle(item.id)} aria-expanded={openId === item.id} aria-controls={`faq-${item.id}`}>`
         - `<span>{item.question}</span>`
         - `<ChevronDown className="fa-chevron" size={20} />`
       - `</button>`
       - `<div className="fa-answer" id={`faq-${item.id}`} role="region" aria-labelledby={`faq-q-${item.id}`}>`
         - `<div className="fa-answer-inner" dangerouslySetInnerHTML={{ __html: item.answer }} />`
       - `</div>`

5. **Toggle function**: `const toggle = (id: string) => setOpenId(prev => prev === id ? null : id)`
